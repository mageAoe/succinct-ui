# scq-succinct-ui

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### 打包命令解释：

--target lib 关键字 指定打包的目录

--name 打包后的文件名字

--dest 打包后的文件夹的名称

### npm 命令

- npm 查看当前源

```
npm get registry
```

- npm 设置淘宝镜像源：

```
npm config set registry http://registry.npm.taobao.org
```

- npm 设置官方源：

```
npm config set registry http://www.npmjs.org
```
